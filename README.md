<h3>HTML</h3>
<p><i>Web sayfalarının içeriğini tanımlamak için kullanılan bir işaretleme dilidir. HTML, web sayfalarına metin, resim, video, bağlantı ve diğer öğeler eklemenizi sağlar.</i></p>

<hr/>

<h3>CSS</h3>
<p><i>Web sayfalarının görünümünü ve düzenini kontrol etmek için kullanılan bir stil dilidir. CSS, web sayfalarının renklerini, boyutlarını, konumlarını ve diğer özelliklerini değiştirmenize olanak tanır.</i></p>

<hr/>

<h3>JAVASCRIPT</h3>
<p><i>Web sayfalarına etkileşimli özellikler eklemek için kullanılan bir programlama dilidir. JavaScript, kullanıcıların web sayfalarıyla etkileşime girmesini sağlayan animasyonlar, formlar ve diğer öğeler oluşturmanıza olanak tanır.</i></p>

<hr/>

<h3>JSP</h3>
<p><i>Java ile yazılmış web sayfaları oluşturmak için kullanılan bir teknolojidir. JSP, HTML, CSS ve JavaScript'in yanı sıra Java kodunu da kullanmanıza olanak tanır.</i></p>
